# 🎵 מוריד זמרשת

הורדת שירים מזמרשת **ישירות לתיקייה** - עם שמות עבריים מלאים!

## 📦 התקנה (פעם אחת)

### שלב 1: ודא ש-Node.js מותקן
```bash
node --version
```

אם לא מותקן, הורד מ: https://nodejs.org

### שלב 2: התקן את הספריות
```bash
cd zemereshet-downloader
npm install
```

זהו! ההתקנה הושלמה ✅

---

## 🚀 שימוש

### כל פעם שאתה רוצה להשתמש:

1. **הפעל את השרת:**
```bash
npm start
```

2. **פתח בדפדפן:**
`http://localhost:3000`

3. **הדבק קישור לשיר:**
   - `https://www.zemereshet.co.il/song.asp?id=5197`

4. **לחץ "הורד ZIP"**

5. **זהו!** הקובץ ZIP יורד עם כל ההקלטות ושמות עבריים מושלמים! 🎉

---

## ✨ מה מקבלים?

קובץ ZIP עם שם השיר:
```
חמישה_כושים_אנו.zip
├── 01 - חמישה כושים אנו - חבורת שהם 2012.mp3
├── 02 - חמישה כושים אנו - להקת הנח"ל 1968.mp3
├── 03 - חמישה כושים אנו - דודו זכאי.mp3
├── ...
└── מידע_על_השיר.txt
```

**כל השמות בעברית מלאה - פשוט פתח את ה-ZIP ותהנה!** 🇮🇱

---

## ⚙️ איך זה עובד?

השרת מכין לך ZIP מושלם:
1. טוען את דף השיר מזמרשת
2. מחלץ את כל ההקלטות (MP3)
3. יוצר ZIP עם כל הקבצים ושמות עבריים מלאים
4. מוסיף קובץ מידע מפורט
5. שולח לך את ה-ZIP להורדה

**ZIP עם שמות עבריים מושלמים!** 💪

---

## 🛑 עצירת השרת

כשסיימת, פשוט לחץ `Ctrl+C` בטרמינל.

## ❓ בעיות?

